如何测试：
创建用户
```
curl -XPOST -H"Content-Type: application/json" http://127.0.0.1:5555/v1/users -d'{"username":"authntest","password":"miniblog1234","nickname":"colin404","email":"colin404@foxmail.com","phone":"18130000000"}'
```
登录
```
curl -XPOST -H "Content-Type: application/json" http://127.0.0.1:5555/login -d '{"username":"authntest","password":"miniblog1234"}'
```
使用登录返回的 token 创建博客

创建博客
```
curl -XPOST -H "Content-Type: application/json" -H "Authorization: Bearer token" http://127.0.0.1:5555/v1/posts -d'{"title":"installation","content":"installation."}'
```
